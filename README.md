## Getting Started

```bash
npm install
# then
npm run dev
```

## How to commit
When you run git commit, you can add the message directly using the -m flag:
```bash
git commit -m "fix: resolve login button alignment issue"
```

See the result on your browser [http://localhost:3000](http://localhost:3000)
